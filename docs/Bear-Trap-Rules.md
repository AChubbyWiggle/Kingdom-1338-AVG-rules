Please keep an eye out for messages changing the Bear trap timings. There may be times where we have overlapping events and may need to adjust.

- R3 and up to start rallies
- At 6:30 minutes remaining everyone can set a rally.
- Use a 10/10/80 formation
- Set formations in advance to speed up rally join time
